# Glucose_project_demo
Demo de um projeto de IA aplicada à saúde focado em medição de glicemia não invasiva, abrangendo aumento de acurácia de sensores, previsão de glicose em tempo real (até 1h à frente) e geração adaptativa de dietas baseada em dados glicêmicos contínuos, com validação rigorosa e modelagem de séries temporais.
